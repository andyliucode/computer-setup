New MacBook Pro Setup

System Preferences
	Sound
		Check ‘Show volume in menu bar’
	Trackpad
		Secondary click
			Select 'Click in bottom right corner'
		Tap to click
			Check ‘Tap to click’
		Scroll & Zoom
			Scroll direction: Inverse
	Accessibility
		Trackpad Options…
			Check ‘Enable dragging’ and select ‘three finger drag’

	Turn off mouse acceleration
		defaults write .GlobalPreferences com.apple.mouse.scaling -1


Software
	Sublime Text
		Setup running from command line:
			mkdir $HOME/bin
			ln -s '/Applications/Sublime Text.app/Contents/SharedSupport/bin/subl' $HOME/bin/subl
			echo 'export PATH=$HOME/bin:$PATH' >> ~/.bashrc
		Copy in preferences and keybinds from files

	Xcode Command Line Tools
	
	Git
		Setup ssh keys
			https://kbroman.org/github_tutorial/pages/first_time.html
		Setup git config
			git config --global user.email "you@example.com"
			git config --global user.name "Your Name"

	Safari
		AdGuard for Safari

	Python3
		Should be installed by Xcode
		pip3 install --user virtualenv
